Digital Annealer Development Kit#

Welcome to the documentation for the Digital Annealer Development Kit (DADK).

Many combinatorial optimization problems can be formulated as quadratic unconstrained binary optimization problems (QUBOs) which then be can solved efficiently on special purpose hardware, so-called Ising machines. A specific Ising machine is Fujitsuâ€™s Digital Annealer , a quantum-inspired technology for solving large and complex combinatorial optimization problems.

The dadk is a python library for creating and manipulating quadratic unconstrained binary polynomials. It also includes a solver which emulates the Digital Annealer annealing algorithm. Furthermore, access to the Digital Annealer services is also possible, provided a valid API key has been assigned to you.
