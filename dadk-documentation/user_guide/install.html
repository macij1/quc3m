Installation#

In order to obtain the DADK SDK you need to download the Digital Annealer Tutorial here . The download contains extensive study material and implementations of business use cases in the Tutorial folder. But more importantly it also contains the DADK Python SDK which is located in the Software folder.

Supported Systems#

DADK is tested on and working on the following systems:

  • Linux

  • Windows 10+

  • Mac (Intel only)

Python#

We recommend using Python 3.10 and setting up a fresh virtual environment, e.g. using venv.

python -m venv dadk-310

DADK#

The library can be found in the folder Software of the Digital Annealer Tutorial and can be installed using pip:

pip install -U <path_to>\dadk-light.tar.bz2

You can a version check to see if the installation worked

import dadk
dadk.DADK_version

Tutorials#

The extensive tutorial material is contained in the Tutorial folder and is best accessed using jupyter notebooks. The notebook Introduction.ipynb gives an overview of the tutorial material.
