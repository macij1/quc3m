Constraints#

Quadratic Unconstrained Binary Optimization means that each constraint has to be formulated as a quadratic binary formula which evaluates to zero for states which fulfill the constraint and which attain a value larger than zero for states which violate the formula. These penalties (also called relaxed constraints) are then added to the objective with appropriate pre-factors in order to express the constrained problem in an unconstrained way.

This approach is problematic for constraints which cannot be directly formulated as penalties. In particular, most inequality constraints require additional binary variables in order to be expressed as quadratic penalties. For many inequality constraints like big-M constraints this results in a very large number of auxiliary variables, enlarging the search space drastically.

From Digital Annealer version 3 onwards, linear binary inequality constraints are supported directly. Moreover, binary quadratic penalties can be separated from the objective, taking away a large part of complicated weighting between penalties and objectives.

Also DADK’s QUBOSolverCPU solver supports linear binary inequality constraints.

We explain how to program inequality constraints in the following.

Example Problem#

Again let var_shape_set be programmed and fixed as in the previous sections:

from dadk.BinPol import *

days = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday']
D = len(days)

shifts = ['Early', 'Late', 'Night']
S = len(shifts)

workers = ['Adam1', 'Bob1', 'Charly1', 'Dave1', 'Earl1',
           'Adam2', 'Bob2', 'Charly2', 'Dave2', 'Earl2',
           'Adam3', 'Bob3', 'Charly3', 'Dave3', 'Earl3',
           'Adam4', 'Bob4', 'Charly4', 'Dave4', 'Earl4']
W = len(workers)

blocked = [('Monday', 'Night', 'Adam1'),
           ('Wednesday', 'Late', 'Charly2'),
           ('Friday', 'Early', 'Earl2')]

constant_bits_x_bits = np.full((D, S, W), -1, dtype=np.int8)
for day, shift, worker in blocked:
    constant_bits_x_bits[days.index(day), shifts.index(shift), workers.index(worker)] = 0

x_bit_array_shape = BitArrayShape(name='x', constant_bits=constant_bits_x_bits)

demand = {
    'Monday': {'Early': 2, 'Late': 5, 'Night': 3},
    'Tuesday': {'Early': 1, 'Late': 5, 'Night': 3},
    'Wednesday': {'Early': 2, 'Late': 5, 'Night': 3},
    'Thursday': {'Early': 1, 'Late': 5, 'Night': 3},
    'Friday': {'Early': 2, 'Late': 5, 'Night': 1}}

one_hot_groups_list = [OneHotGroup(('x', d, s, None)) for d, day in enumerate(days) for s, shift in enumerate(shifts) if demand[day][shift] == 1]

y_bit_array_shape = BitArrayShape(name='y', shape=(W,))

var_shape_set = VarShapeSet(x_bit_array_shape,
                            y_bit_array_shape,
                            one_hot_groups=one_hot_groups_list)
BinPol.freeze_var_shape_set(var_shape_set)

Inequalities for Workers’ Shifts per Week#

Suppose each worker may only work up to four shifts a week, i.e., \(\sum_{d\in \mathcal{D}}\sum_{s\in \mathcal{S}}x_{d,s,w} \leq 4\ \forall w\in \mathcal{W}.\) Those inequalities are programmed in DADK as follows:

upper_shift_limit = 4
inequalities = []
for w, worker in enumerate(workers):
    tmp_qubo = BinPol().add_term(-upper_shift_limit)
    for d, day in enumerate(days):
        for s, shift in enumerate(shifts):
            tmp_qubo.add_term(1, ('x', d, s, w))
    inequalities.append(Inequality(qubo=tmp_qubo, lambda_value=1000))

Note that for each \(w\in \mathcal{W}\), the corresponding constraint is formulated as \(\sum_{d\in \mathcal{D}}\sum_{s\in \mathcal{S}}x_{d,s,w} - 4 \leq 0\), i.e., as an inequality with left-hand side less or equal zero. The left-hand side is programmed as a BinPol object and provided to Inequality along with a lambda_value which weights the particular inequality against other constraints later during optimization. All inequality constraints are collected in a list inequalities which is later provided to the solver along with the objective and the penalties.

The latter are programmed as before but can be separated from the objective from Digital Annealer version 3.

Optimization Objectives#

For the objective to minimize the number of workers the program looks as follows:

var_shape_set = VarShapeSet(x_bit_array_shape,
                            y_bit_array_shape,
                            one_hot_groups=one_hot_groups_list)
BinPol.freeze_var_shape_set(var_shape_set)

weight = 2000
penalty_qubo = BinPol()
for d, day in enumerate(days):
    for s, shift in enumerate(shifts):
        penalty_qubo.add_exactly_n_bits_on(n=demand[day][shift],
                                           bits=[('x', d, s, w) for w, worker in enumerate(workers)],
                                           scalar=weight)
for w, worker in enumerate(workers):
    for d, day in enumerate(days):
        penalty_qubo.add_most_1_bit_on(bits=[('x', d, s, w) for s, shift in enumerate(shifts)],
                                       scalar=weight)

for w, worker in enumerate(workers):
    for d, day in enumerate(days):
        for s, shift in enumerate(shifts):
            penalty_qubo.add_term(weight, ('x', d, s, w))
            penalty_qubo.add_term(-weight, ('x', d, s, w), ('y', w))

weight = 1800
qubo = BinPol()
for w, worker in enumerate(workers):
    qubo.add_term(weight, ('y', w))

For the objective to spread the work evenly among all workers, the code looks like this:

var_shape_set = VarShapeSet(x_bit_array_shape,
                            one_hot_groups=one_hot_groups_list)
BinPol.freeze_var_shape_set(var_shape_set)

weight = 2000
penalty_qubo = BinPol()
for d, day in enumerate(days):
    for s, shift in enumerate(shifts):
        penalty_qubo.add_exactly_n_bits_on(n=demand[day][shift],
                                           bits=[('x', d, s, w) for w, worker in enumerate(workers)],
                                           scalar=weight)
for w, worker in enumerate(workers):
    for d, day in enumerate(days):
        penalty_qubo.add_most_1_bit_on(bits=[('x', d, s, w) for s, shift in enumerate(shifts)],
                                       scalar=weight)

nr_shifts_per_worker = sum([demand[day][shift] for d, day in enumerate(days) for s, shift in enumerate(shifts)]) / W
weight = 400
for w, worker in enumerate(workers):
    qubo_tmp = BinPol()
    qubo_tmp.add_term(nr_shifts_per_worker)
    for d, day in enumerate(days):
        for s, shift in enumerate(shifts):
            qubo_tmp.add_term(-1, ('x', d, s, w))
    qubo_tmp.power2()
    qubo.add(qubo_tmp, weight)
